# Awesome Skills

This repository collects reusable Codex skills. Each skill lives in its own
subdirectory under `skills/` and includes its own documentation.

## Skills

| Skill | Description |
| --- | --- |
| [coinglass-api-decrypt](skills/coinglass-api-decrypt/README.md) | Analyze and decrypt encrypted CoinGlass web API responses. |

## Repository Layout

```text
skills/
  <skill-name>/
    SKILL.md
    README.md
    scripts/
```

When adding a new skill, create a `README.md` inside the skill directory and add
it to the table above.
